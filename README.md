# chrome-cli 
chrome插件脚手架。

安装

```
npm i chrome-plugins-cli
```



命令说明

```
chrome-plugins-cli init
```

| 命令 | 说明       | 描述                                                         |
| ---- | ---------- | ------------------------------------------------------------ |
| init | 工程初始化 | 参数包括projectName(项目名称)，description(项目描述)，author(作者) |
|      |            |                                                              |
|      |            |                                                              |
